# LoopLM-Style Prototype Track

This directory is for small, explicit proxy experiments inspired by looped or
latent-reasoning papers.

## Current Scope

- baseline against the unmodified Gemma PT backend
- minimal adapter-style prototype only
- loop depth ablations such as `K=1/2/4`
- quality, latency, and memory captured together

## Expected Artifacts

- experiment plans in markdown
- run metadata in `research/looplm/runs/`

## Guardrail

Do not describe this track as a full reproduction unless the evidence actually
supports that claim.
